A fully responsive testimonials plugin with bootstrap and limit for Slides - Grid with Columns By Innvonix Technologies LLP.

== Description ==
Testimonial Slider and Grid view with Mobile and Tablet Responsive using simple Shortcodes. 

Innvonix Testimonials is a WordPress Plugin with Slider and Grid Layouts with easy setup with shortcodes. Modern design and Faster loading with quick response Light weight, Touch Supported,  WordPress Plugin. Easily view testimonials in one or two columns. We will come soon with new themes and layouts.
 
<strong>A few notes about Innvonix Testimonials Slider:</strong>
 
*   Shortcode [testimonials-slider] & [testimonials-grid] with "limit", "Order", "Column", "Arrows" attributes.
*   Fully customizable Slider and Grid Layouts
*   Testimonials, testimonials slider, slider, quote, silder testimonials, comments, client testimonials
*	Requires at least WordPress 4.4 version
*   Tested Upto WordPress 4.4 to WordPress 4.8 
 
== Installation ==
 
Installation Steps are as follow.
 
 
1. Upload `innvonix-testimonials.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Or

1. Go throught Plugins > Add New > Search Invonix Testomonials and Install 
2. If you have already downloaded the plugin go through Plugins > Add New > Upload and select the innvonix-testimonials.zip after installation activate plugin.
